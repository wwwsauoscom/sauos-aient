# SAUOS - AI 电脑全自动化系统

## 产品概述

**SAUOS**（Screen Automation Universal Operating System）是新一代 AI 驱动的桌面自动化解决方案。通过结合大语言模型的视觉理解能力与精准的系统控制技术，SAUOS 实现了"自然语言操控电脑"的愿景——用户只需描述任务目标，AI 即可自主完成从屏幕分析到操作执行的全流程。

---

## 核心价值

### 1. 自然语言驱动
告别复杂的脚本编写和手动操作记录。直接用自然语言描述任务：
> "打开浏览器，搜索市场分析报告，下载第一个PDF文件"

AI 自动理解意图，规划路径，逐步执行。

### 2. 视觉智能识别
基于多模态大模型的视觉理解能力：
- 实时分析屏幕内容
- 智能识别 UI 元素位置
- 动态适应界面变化
- 无需预置坐标或图像模板

### 3. 多模型架构
灵活支持多种 AI 服务商，一键切换：

| 类型 | 支持平台 |
|------|----------|
| 国际 | OpenAI GPT-4o、Claude Sonnet |
| 国内 | 阿里百炼（通义千问）、MiniMax、火山引擎（豆包）、DeepSeek、智谱GLM、月之暗面Kimi |
| 本地 | Ollama（支持 Llama、Qwen 等开源模型） |

### 4. 企业级安全
- 支持本地私有化部署
- 数据不出内网
- 完全掌控 AI 模型选择
