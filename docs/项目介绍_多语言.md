## 中文 (Chinese)

### SAUOS - AI驱动的智能桌面自动化系统

**官网**: https://sauos-aient.sauos.net  
**GitHub**: https://github.com/sauos-dev/sauos-aient

#### 简介
SAUOS (Screen Automation Universal Operating System) 是一个基于人工智能的桌面自动化系统。通过大语言模型和视觉理解能力，SAUOS 能够让计算机像人一样"看"屏幕、"理解"界面、"执行"操作，实现自然语言驱动的计算机操作自动化。

#### 核心特性
- **视觉理解** - 实时截屏分析，精准识别UI元素、文本内容、按钮位置
- **智能决策** - 基于LLM的任务规划，自动分解复杂操作步骤
- **精准操控** - 鼠标点击/拖拽、键盘输入、热键组合、窗口管理
- **多模型支持** - OpenAI、Claude、阿里百炼、DeepSeek等8+大模型一键切换
- **私有部署** - 支持Ollama本地模型，数据不出企业内网

#### 应用场景
- 办公自动化：批量处理文档、数据录入、报表生成
- 软件测试：UI自动化测试、回归测试
- 数据采集：网页信息提取、表单自动填写
- 流程机器人：跨系统操作、重复任务自动化

## English

### SAUOS - AI-Powered Intelligent Desktop Automation System

**Website**: https://sauos-aient.sauos.net  
**GitHub**: https://github.com/sauos-dev/sauos-aient

#### Introduction
SAUOS (Screen Automation Universal Operating System) is an AI-based desktop automation system. By leveraging large language models and visual understanding capabilities, SAUOS enables computers to "see" screens, "understand" interfaces, and "execute" operations like humans, achieving natural language-driven computer automation.

#### Core Features
- **Visual Understanding** - Real-time screen analysis, accurate recognition of UI elements, text content, and button positions
- **Intelligent Decision Making** - LLM-based task planning, automatic decomposition of complex operations
- **Precise Control** - Mouse clicking/dragging, keyboard input, hotkey combinations, window management
- **Multi-Model Support** - One-click switching between 8+ LLMs including OpenAI, Claude, Alibaba Bailian, DeepSeek
- **Private Deployment** - Support for Ollama local models, data never leaves your network

#### Use Cases
- Office Automation: Batch document processing, data entry, report generation
- Software Testing: UI automation testing, regression testing
- Data Collection: Web information extraction, automatic form filling
- Process Robots: Cross-system operations, repetitive task automation

## 한국어 (Korean)

### SAUOS - AI 기반 지능형 데스크톱 자동화 시스템

**웹사이트**: https://sauos-aient.sauos.net  
**GitHub**: https://github.com/sauos-dev/sauos-aient

#### 소개
SAUOS(Screen Automation Universal Operating System)는 AI 기반 데스크톱 자동화 시스템입니다. 대규모 언어 모델과 시각적 이해 기능을 활용하여 SAUOS는 컴퓨터가 사람처럼 화면을 "보고", 인터페이스를 "이해하고", 작업을 "실행"할 수 있게 하여 자연어 기반 컴퓨터 자동화를 구현합니다.

#### 핵심 기능
- **시각적 이해** - 실시간 화면 분석, UI 요소, 텍스트 내용, 버튼 위치의 정확한 인식
- **지능적 의사결정** - LLM 기반 작업 계획, 복잡한 작업의 자동 분해
- **정밀 제어** - 마우스 클릭/드래그, 키보드 입력, 단축키 조합, 창 관리
- **다중 모델 지원** - OpenAI, Claude, 알리바바 바이리안, DeepSeek 등 8개 이상의 LLM 원클릭 전환
- **프라이빗 배포** - Ollama 로컬 모델 지원, 데이터가 기업 네트워크를 벗어나지 않음

#### 활용 사례
- 사무 자동화: 대량 문서 처리, 데이터 입력, 보고서 생성
- 소프트웨어 테스트: UI 자동화 테스트, 회귀 테스트
- 데이터 수집: 웹 정보 추출, 자동 양식 작성
- 프로세스 로봇: 크로스 시스템 작업, 반복 작업 자동화

- **Private Bereitstellung** - Unterstützung für Ollama lokale Modelle, Daten verlassen nie Ihr Netzwerk

- Büroautomatisierung: Stapelverarbeitung von Dokumenten, Dateneingabe, Berichtserstellung

## Русский (Russian)

### SAUOS - Интеллектуальная система автоматизации рабочего стола на базе ИИ

**Веб-сайт**: https://sauos-aient.sauos.net  
**GitHub**: https://github.com/sauos-dev/sauos-aient

#### Введение
SAUOS (Screen Automation Universal Operating System) — это система автоматизации рабочего стола на основе искусственного интеллекта. Используя большие языковые модели и возможности визуального понимания, SAUOS позволяет компьютерам "видеть" экраны, "понимать" интерфейсы и "выполнять" операции как люди, обеспечивая автоматизацию компьютера на естественном языке.

#### Основные функции
- **Визуальное понимание** - Анализ экрана в реальном времени, точное распознавание элементов UI, текстового содержимого и позиций кнопок
- **Интеллектуальное принятие решений** - Планирование задач на основе LLM, автоматическая декомпозиция сложных операций
- **Точное управление** - Клики/перетаскивание мыши, ввод с клавиатуры, комбинации горячих клавиш, управление окнами
- **Поддержка нескольких моделей** - Переключение одним кликом между 8+ LLM, включая OpenAI, Claude, Alibaba Bailian, DeepSeek
- **Частное развертывание** - Поддержка локальных моделей Ollama, данные никогда не покидают вашу сеть

#### Сценарии использования
- Автоматизация офиса: Пакетная обработка документов, ввод данных, создание отчетов
- Тестирование ПО: Автоматизированное UI-тестирование, регрессионное тестирование
- Сбор данных: Извлечение информации из веба, автоматическое заполнение форм
- Процессные роботы: Межсистемные операции, автоматизация повторяющихся задач
